Free Chrome Extension

Salesforce Manager

Browse, compare, import, and export Permission Set object & field permissions — right from your browser. No setup required.

Add to Chrome — It's Free
Features

Everything a Salesforce admin needs

One extension. Two powerful managers. No Connected App, no OAuth setup, no extra logins.

🔒

Object Permissions Matrix

Select multiple Permission Sets and view a side-by-side matrix of C / R / U / D / VA / MA permissions per object. Filter by permission type or object name.

📋

Field Permissions Matrix

Drill into any object's field-level security across Permission Sets. See Read / Edit access per field in one scrollable table.

⇓

Export to Excel

Download the full permissions matrix as a structured .xlsx file for offline review, auditing, or sharing with stakeholders.

⇑

Import & Apply Changes

Upload a modified .xlsx file, preview a delta of what changed, then apply the changes directly to your org in one click.

🔍

Smart Filtering

Filter objects and fields by permission type. Dependency rules automatically enforce Salesforce's permission hierarchy (e.g. Edit requires Read).

🔒

No Login Required

Uses your existing Salesforce browser session automatically. No Connected App, no OAuth, no client ID to configure.

🌞

Tab Colours for Multiple Orgs

Automatically colour-codes the browser tab favicon for each org so you can tell them apart at a glance — no more squinting at URLs. Customise colours, toggle the feature on or off, and import/export your colour settings from within the extension.

🔒

Your data never leaves your browser

All API calls go directly to your Salesforce org. No data is sent to any external server, tracked, or stored beyond your current browser session. Read the full privacy policy.

How it works

Up and running in seconds

No configuration needed. Just install and click.

Install the extension

Add Salesforce Manager from the Chrome Web Store. No sign-up required.

Log in to your Salesforce org

Navigate to any Salesforce org tab in Chrome as you normally would.

Click the extension icon

The Salesforce Manager page opens in a new tab, already connected to your org.

Select Permission Sets and explore

Search and multi-select Permission Sets, then switch between Object and Field Permissions tabs to explore, export, or import.
